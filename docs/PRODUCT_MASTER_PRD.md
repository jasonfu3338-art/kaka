# Kaka 产品总纲 PRD

## 项目定位

Kaka 是面向语音厅生态的模板编辑平台。产品核心不是自由设计工具，而是模板消费工具：用户从模板市场选择模板，在移动端通过快捷编辑完成内容替换、样式微调和图片导出。

第一版坚持 Mobile First，优先保证语音厅用户在手机上的模板浏览、模板详情、快捷编辑和导出链路可用。

## 核心原则

- 模板本质是 Template JSON，不是单张静态图片。
- 编辑器统一使用 Fabric.js 渲染和承载画布。
- 快捷编辑由 `editableFields` 自动驱动，不为每个模板单独写表单。
- 第一版不做视频时间轴、复杂动画、复杂自由拖拽、数据库保存和 AI 能力。
- 单文件尽量不超过 300 行，组件保持可复用。

## 目标用户

- 语音厅房主、厅管、运营人员。
- 需要快速制作语音厅厅头、厅背景、厅封面、冠歌卡、歌单、海报等素材的内容生产者。
- 不熟悉专业设计软件，但希望通过模板快速替换文字和图片的用户。

## 核心模块

### 首页模板市场

展示模板分类、推荐模板、精品模板和快捷入口。当前已有首页 UI、模板卡片、分类筛选与移动端布局基础。

### 模板详情页

展示模板预览、标签、基础信息、相关模板和使用入口。当前已有模板详情页 UI。

### 编辑器

编辑器是核心工作区，当前已经从 UI 阶段进入 Editor Logic 阶段。第一阶段只打通 Template JSON 到 Canvas 和快捷编辑同步，不实现完整编辑器。

当前目标链路：

```txt
Template JSON
↓
编辑器读取
↓
Canvas 显示
↓
快捷编辑自动生成表单
↓
修改表单
↓
同步更新 Canvas
```

### 快捷编辑

快捷编辑是 P1 的核心体验。每个模板通过 `editableFields` 声明可编辑字段，编辑器自动生成输入框、文本域或图片替换入口。

当前支持字段类型：

- `text`
- `textarea`
- `image`

### 图层系统

图层系统用于展示 Template JSON 中的 layer 列表，当前只作为基础图层列表展示和后续编辑能力入口。

第一阶段支持图层类型：

- `text`
- `image`
- `decoration`

### 字体与艺术字

字体属于 P1；艺术字属于 P2。当前仅保留 UI 与数据结构扩展空间，不实现完整字体管理和艺术字生成。

### 后台管理

后台管理用于后续上传模板、管理模板 JSON、素材和分类。当前阶段不实现后台功能。

## 版本范围

### P1

- 模板市场
- 模板详情
- 快捷编辑
- 图片替换
- 字体
- 导出 PNG/JPG
- 后台上传模板

### P2

- 艺术字
- Repeat Block
- 二维码
- 我的素材

### P3

- 动态模板
- 视频背景
- MP4 导出
- AI 能力

## 当前阶段范围

当前阶段为 Editor Logic Phase，只实现：

- Template JSON 类型定义
- Layer 类型定义
- editableFields 类型定义
- Mock Template Loader
- 基础 Canvas 渲染
- 快捷编辑自动生成表单
- 表单修改同步 Canvas

当前明确禁止：

- 复杂拖拽
- 复杂缩放
- 撤销重做
- 数据库保存
- 真实保存
- 视频
- AI
- 时间轴
